Looking for Speak Into Space?

Speak Into Space was a PennApps fall 2019 project that used to be running here. Unfortunately, servers with GPUs are pretty expensive to maintain, so we've taken the service down. If you'd like, you can check out our Github repository or our Devpost submission.
